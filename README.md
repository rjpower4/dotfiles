# Rolfe Power's Dotfiles

## Repository Contents

The following programs (listed alphabetically) have configurations in this repository:

- Alacritty
- Bash
- Doom Emacs
- Matplotlib
- Neovim
- Tmux
- Wget

Additionally, a set of scripts is contained in `local/scripts`.

## Installation

To install clone this repository and change directories

``` sh
git clone https://www.github.com/rjpower4/dotfiles.git dotfiles
cd dotfiles
```

Then, run the `setup.sh` script at the repository's top level

``` sh
./setup.sh
```
